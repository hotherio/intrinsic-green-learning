# `igl.viz`

Optional plotting helpers. Requires the ``[viz]`` extra — importing any
submodule of ``igl.viz`` raises [`IGLDependencyError`][igl.IGLDependencyError]
when ``matplotlib`` is missing.

::: igl.viz.dimension_curve.plot_dimension_curve
    options:
      show_root_heading: true
      show_source: false

::: igl.viz._matplotlib.require_matplotlib
    options:
      show_root_heading: true
      show_source: false
